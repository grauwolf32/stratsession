◆ Big-Picture Themes

Nine macro-level themes that defined the OSS security tooling landscape over the past 12 months.

1

AI-assisted vulnerability discovery moved from demo to delivery

DARPA's AIxCC finals at DEF CON 33 awarded $8.5M. The seven finalist Cyber Reasoning Systems were all released under OSI-approved licenses — the first end-to-end open-source vuln-discovery-and-patching systems usable from a developer laptop. 18 real bugs were disclosed during finals.

AIxCC Buttercup Team Atlanta Seclab Taskflow
2

The “agentic security” wave hit both red and blue

Promptfoo, GARAK (NVIDIA), PyRIT (Microsoft), and DeepTeam crystallized as the de-facto stack for LLM red-teaming. Tooling for agent abuse — Bedrock Agent takeover, Copilot prompt injection, MCP-server interception — was the most-discussed novel attack class.

Promptfoo GARAK PyRIT DeepTeam
3

Kubernetes security split into two layers

Detection converged around CNCF projects (Falco, Tetragon, Tracee), while configuration/posture fragmented across CEL-based scanners (Spotter), entitlement analyzers (KIEMPossible), and admission engines (Kyverno, Kubewarden).

Falco Tetragon Spotter Kyverno
4

Supply chain went baseline

SLSA 1.1, GUAC 0.12, Sigstore, OWASP CycloneDX 2.0 preview, and the new Sunshine SBOM visualizer reset expectations: SBOM + signed provenance is no longer “leading-edge”, it is the floor.

SLSA 1.1 GUAC Sigstore CycloneDX 2.0 Sunshine
5

C2 frameworks consolidated; evasion specialized

Empire 6.0 (Go agents, marketplace), C4 (WASM cross-OS plugins), Beaconator, BOAZ (multi-layer EDR evasion) — the offensive tooling market has matured into composable, language-agnostic stacks.

Empire 6.0 C4 BOAZ Beaconator
6

Hardware/firmware attacks dominated academic cons

ReVault (Dell ControlVault3 on 100+ models, Hexacon), EntrySign (custom x86 microcode, OffensiveCon), Cypherock X1 buffer overflow, NVIDIA CUDA driver 0-days, Apple SEP teardown — all from exploitation-focused conferences.

ReVault EntrySign Cypherock X1
7

CI/CD pipeline compromise became tier-1 risk after TeamPCP

A single actor chained GitHub Actions workflow injection to ship malicious updates through Trivy, LiteLLM, KICS, Telnyx, and dozens of npm packages. The OSS response shipped within three months: SmokedMeat, Plumber, Brutus.

SmokedMeat Plumber Brutus Whooli
8

OSS forensics removed the external dependency requirement

CERT UEFI Parser (CMU/SEI, NDA-free UEFI/installer/PE parsing) and mquire (Trail of Bits, Linux memory forensics without external debug symbols via kernel-embedded BTF + Kallsyms) both ship in 2026.

CERT UEFI Parser mquire
9

OSS defensive ops caught up with offensive

Little Snitch for Linux closes the per-host firewall gap. Allama closes the OSS SOAR gap. Betterleaks closes the entropy-based-FP gap in secret scanning. cloud-audit closes the “PMapper is dead” gap.

Little Snitch for Linux Allama Betterleaks cloud-audit
