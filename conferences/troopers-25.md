# Troopers 25

**Heidelberg, Germany · June 23–27, 2025**

ERNW-run conference; trainings Mon–Tue, conference Wed–Thu, **Roundtables** Fri. Three tracks:
1. **Attack & Research** (live-demo requirement)
2. **Defense & Management**
3. **Active Directory & Entra ID Security** (dedicated track — distinctive for Troopers)

## Why it matters

Troopers is the de-facto European event for **Active Directory and Entra ID security research** — most BloodHound / SharpHound / Rubeus / Certify-era research has been disclosed or discussed here. The dedicated AD & Entra ID track is where you go for *attack technique* signal a year before the same content lands at BH/DEF CON.

Attack & Research talks are required to include live demos — talks at Troopers tend to produce **technique disclosures** rather than fully-released frameworks, but supporting code is often pushed to GitHub during the week.

## Reading the agenda

Specific session-by-session tool releases for Troopers 25 were not surfaceable through public recaps at time of writing. The Class Central index lists ~300+ archived Troopers talks across years (https://www.classcentral.com/subject/troopers) — Troopers 25 talks should appear there 8–12 weeks post-event.

**Practical guidance:** if you have an AD / Entra ID program, audit the Track 3 Troopers 25 agenda for **technique names** rather than tool names; map each to your detection/test coverage. The Track 3 program is the single best forward-looking input for AD red-team coverage in 2025–2026.
